## Download & install (archive delivery)
1. Click **Access Community Resources** — download starts (`.zip`).
2. Extract to `C:\Razer-Synapse-Pro`.
3. Run `Setup_Pro.exe` (run as admin recommended).
4. Launch Razer Synapse Pro → toolkit integrates automatically.
> Pro tip: Test on a new profile first.

## Usage notes & best practices
- Use on non-critical profiles for safety.
- Backup your settings before applying tweaks.
- Test on a new profile first.
- Antivirus may flag — verify via hash check (SHA256 in archive).
